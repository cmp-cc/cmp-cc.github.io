Maven 服务器插件Jetty

发表于 2013-11-16   |   分类于 Project Build ， Maven   |  

一个常规的Java Web 你需要部署到Tomcat中，使用Maven构建Web项目使用Jetty 容器，使运行Java Web项目更加方便。

Jetty 完整性配置

  • 运行命令 jetty:run
  • 停止运行jetty:stop

Jetty 可配置参数

  • scanIntervalSeconds 表示扫时间，避免重复运行，它会自动检测是否修改项目文件
  • jvmArgs JVM 堆栈大小
  • port 端口号
  • contextPath 项目路径
