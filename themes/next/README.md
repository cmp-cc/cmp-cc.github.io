# NexT

* NexT 是内容生成系统 Hexo 的一个主题项目。
* NexT Github 项目地址 https://github.com/iissnan/hexo-theme-next
* NexT 整合部分有用的插件。


有时间会对这个主题（NexT）进行简单的二次开发，使它更加趋向于我的风格。 哈哈~~~


NexT 主旨在于简洁优雅且易于使用，所以首先要尽量确保 NexT 的简洁易用性。

NexT is built for easily use with elegant appearance. First things first, always keep things simple.
